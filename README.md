# FlickrTestTask

### Tech
* Retrofit 2 (Fetch data)
* GoogleMap
* Glide (Load image from INTERNET)
* Gson (Working with Json)
* Paper (for Cache)
* Recycler and CardView (Display Item) 
* SwipeRefreshLayout (for refresh data)
* SpotsDialog (moving spots progress)
